[ENGLISH VERSION](./README-en.md)

## 実行するには
1. LINE BOTのアカウントを作成する
1. `npm install` または `yarn install` を実行する
1. `.env` ファイルを作成する  
    ```
    CHANNEL_SECRET=XXXXXXXXX
    CHANNEL_ACCESS_TOKEN=XXXXXXXXXXXXX
    ```
1. `npm start` または `yarn start` でサーバーを開始する
